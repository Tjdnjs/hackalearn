#### 주제 선정 배경

대학 생활 중 좋은 프로그램이 많지만, 그 프로그램을 참여하기 위해 어떠한 준비과정을 거쳐야하는 지에 대한 정보는 얻기 힘들다. 하여, 해당 경험이 있는 멘토를 통해 원하는 정보를 얻을 수 있는 사이트 제작을 기획하게 되었다.

#### 기능

1. 로그인 / 로그아웃
2. 게시글 작성 / 열람 
3. 댓글 작성 / 열람
4. 게시글 카테고리 분류 자동화
5. 게시글 / 댓글 작성 권한을 로그인 된 사용자에게만 부여
